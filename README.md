# 🧮 Interactive Math Game (MATLAB GUI)
This is an educational tool designed for students to practice simple arithmetic problems in a fun and interactive way.

## ✨ Features
- **Random Questions:** Generates simple math problems automatically.
- **Instant Feedback:** Tells the student if the answer is correct or not.
- **Smart Design:** Clean UI using professional colors and fonts.

## 🚀 How to Run
1. Open MATLAB.
2. Run the `filename.m` file (استبدلي filename باسم ملفك).
3. Start solving!
